Description

Questa ontologia ha l'obiettivo di fornire una rappresentazione generale sulle principali tematiche trattate dalla letteratura, con il particolare obiettivo di rendere possibile il suggerimento di nuove da parte di esperti o amatori.

GrandeSaga c

IRI http://www.semanticweb.org/owl/owlapi/turtle#GrandeSaga
Description

Una grande saga è una narrazione estesa e complessa che si sviluppa su molteplici opere letterarie, spesso seguendo diversi gruppi di personaggi o intere famiglie, con una trama che si estende su generazioni o eventi di ampia portata.
Sub Class Of Saga c

Trilogia c

IRI http://www.semanticweb.org/owl/owlapi/turtle#Trilogia
Description

Una trilogia è una saga composta da tre opere letterarie che si sviluppano attraverso una trama continuativa, generalmente seguendo lo stesso gruppo di personaggi o un arco narrativo unico.

Funziona solo se sono definiti con enumerazione
